# 🎬 Movie Review and Recommendation System (Without SQL)

A **Python-based movie review and recommendation system** built without
SQL databases. This project demonstrates how to handle movie data,
collect user reviews, and generate personalized recommendations using
in-memory data structures and machine learning concepts.

------------------------------------------------------------------------

## 📌 Features

-   **Movie Reviews**: Users can add and view reviews for movies.
-   **Recommendation Engine**: Suggests movies based on user preferences
    and ratings.
-   **Search Functionality**: Find movies by title or genre.
-   **No SQL Dependency**: Uses Python dictionaries/lists instead of SQL
    databases.
-   **Simple UI**: Command-line interface for easy interaction.

------------------------------------------------------------------------

## 🛠️ Tech Stack

-   **Python** (Core language)
-   **CSV** (Data Storage)
